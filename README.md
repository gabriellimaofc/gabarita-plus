# gabarita-plus
🎓 Gabarita+

Plataforma inteligente de treinamento para o ENEM.

O Gabarita+ é um sistema completo de estudos desenvolvido para ajudar estudantes a evoluírem através de questões, simulados, análises de desempenho e revisão inteligente.

🚀 Funcionalidades
✅ Banco de questões
✅ Filtros avançados
✅ Simulados estilo ENEM
✅ Caderno de erros
✅ Dashboard de desempenho
✅ Sistema de revisão inteligente
✅ Favoritos
✅ Responsivo para desktop, tablet e mobile
🧠 Tecnologias
Front-end
React
Next.js
TailwindCSS
Back-end
Java
Spring Boot
JWT Authentication
Banco de dados
PostgreSQL
Redis
📊 Objetivo do projeto

Transformar o estudo para o ENEM em uma experiência moderna, inteligente e personalizada.
